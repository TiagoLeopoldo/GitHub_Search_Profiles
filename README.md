# 📌 Github Search Profile ![GitHub](src/assets/imgs/icon-gh.svg)

## 🔹 Descrição  
Este projeto permite **buscar perfis do GitHub** e visualizar informações como avatar, nome de usuário e bio.

- ⚛️ React  
- 🎨 TailwindCSS  
- ⚡ Vite
